üìå KPI‚Äôs REQUIREMENT :
The business wants to calculate these five key metrics to understand its performance:

1. Total Revenue:
Definition: The total money earned from all pizza orders.
Calculation: Add the price of every pizza order.

2. Average Order Value (AOV):
Definition: The average amount of money a customer spends per order.
Calculation:
\text{Average Order Value} = \frac{\text{Total Revenue}}{\text{Total Orders}}

3. Total Pizzas Sold:
Definition: Total number of pizzas sold (sum of all pizzas ordered).

4. Total Orders:
Definition: Total number of customer orders placed.

5. Average Pizzas Per Order:
Definition: The average number of pizzas in each order.
Calculation:
\text{Average Pizzas Per Order} = \frac{\text{Total Pizzas Sold}}{\text{Total Orders}}
